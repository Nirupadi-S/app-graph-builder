# App Graph Builder

**LIVE DEMO:**
https://app-graph-builder-5jha7u7tw-nirupadi-s-projects.vercel.app

## Setup

npm install

npm run dev

## Tech Stack

* React
* TypeScript
* Vite
* React Flow
* Zustand
* TanStack Query
* Tailwind CSS

## Features

* Multi-app graph visualization
* React Flow canvas
* Dotted background
* Node Inspector
* Config and Runtime tabs
* Editable node name
* Synced slider and numeric input
* Delete node with keyboard shortcut
* Mobile responsive inspector drawer
* Mock API integration
* Query caching with TanStack Query

## Scripts

npm run dev

npm run build

npm run preview

npm run lint

npm run typecheck

## Key Decisions

* Zustand manages UI state
* TanStack Query manages server state
* React Flow handles graph rendering
* Mobile drawer used for small screens

## Known Limitations

* Mock data only
* No backend persistence
* Error state is simulated
